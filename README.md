Creating Views

## Objective
This task demonstrates how to create and use SQL Views for data abstraction and reusable query logic.

## Concepts Covered
- Simple View
- View with JOIN
- Aggregate View
- Dropping a View
- Data Abstraction

## Tools Used
- DB Browser for SQLite

## Description
Different types of views were created to simplify complex queries and restrict direct table access. Views help in abstraction, security, and reusable SQL logic.
